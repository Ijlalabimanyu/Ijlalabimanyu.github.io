Application YUK Curhat!

Aplikasi YUK Curhat berbasis Mobile Apps merupakan sebuah platform yang tepat untuk digunakan para penderita mental health. Aplikasi YUK Curhat ini bertujuan membantu para masyarakat dari usia (15 â€“ 24 tahun) agar dapat meluapkan pikiran dan isi hati untuk menghindarkan beban pikiran atau mental down hanya karena takut memberikan sebuah stigma negatif terhadap lawan bicara dan mendapat sebuah perlakuan perundungan (bullying) yang akan berdampak kepada penderita semakin buruk serta dapat menyakiti dirinya sendiri (self harm). Aplikasi YUK Curhat akan menyediakan Admin yang memiliki spesialis dan ditangani secara profesional untuk menyelesaikan masalah yang ada. Para penderita juga dapat berinteraksi dengan menceritakan dan memberikan pendapat di dalam Community yang telah disediakan pada aplikasi YUK Curhat tersebut. Selain itu juga, didalam YUK Curhat para pengguna dapat memilih ingin mencerita dengan mengetik secara panjang atau lebih memilih dengan menggunakan suara agar lebih ekspresif dalam mengutarakan apa yang sedang dialami. Pada aplikasi ini sangat meminimalisir para pengguna untuk memberikan informasi secara terang-terangan agar terhindarkan dari penyalahgunaan informasi oleh para pengguna orang lain serta menjaga keamanan para pengguna dalam menceritakan masalah yang sedang dialaminya.
Onboarding Page

Memberikan gambaran aplikasi YUK Curhat bagi user yang baru saja bergabung sebelum masuk ke halaman Sign In

Tampilan Awal User Register dan Sign In

User harus melakukan registrasi pada halaman Create Account dengan memasukkan data berupa username, tanggal lahir, nomor telepon, email, dan password

Home Page

Terdapat event-event yang diadakan seperti webinar, kemudian terdapat fitur search admin yang dapat digunakan oleh user untuk mencari admin yang diinginkan.

Booking Page

User dapat melihat informasi admin berupa umur, gender, dan waktu yang tersedia. Klik button booking untuk mengatur janjian dengan admin. User diminta mengisi informasi terkait pertemuan dengan admin.

Fitur Community

User dapat berbagi pengalaman dan melihat cerita yang dibagikan oleh user lainnya, sehingga mereka mendapatkan feedback dari pengalaman yang dialami.

Notification Page

Memuat pemberitahuan yang ditujukan kepada user tentang event-event yang sedang diadakan, pemberitahuan dari admin, ataupun aktivitas dari user lainnya

Profile Page

Memuat informasi akun user dan aktivitas seperti story atau podcast yang pernah dibagikan. Pada profile juga terdapat fitur history, help, setting, dan logout.
